# Changeset Branches
Redmine patches Plugin.This plugin contains fixes for redmine.

## How to install
1. Download sources
2. Put them in redmine\plugins directory

## Fixes:
1. Updated CSS for dropdowns on filter block.
2. Fixed 'tags' column when exporting tickets to CSV.
